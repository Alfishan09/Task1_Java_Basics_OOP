## 🚀 Task 1: Java Basics and Object-Oriented Programming Fundamentals

## 📌 Overview

This project was developed as part of the Alfido Tech Java Developer Internship. It focuses on the implementation of fundamental Java programming concepts and Object-Oriented Programming (OOP) principles through simple and structured Java programs.

The project demonstrates practical applications of conditional statements, loops, arrays, classes, inheritance, polymorphism, and encapsulation.

- Understand the fundamentals of Core Java programming.
- Implement basic programming constructs such as conditions, loops, and arrays.
- Demonstrate Object-Oriented Programming concepts using Java.

- Strengthen problem-solving and logical thinking skills.
